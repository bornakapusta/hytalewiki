## Templates

### Plugin Template

Starter project with recommended structure:

```
my-plugin/
├── build.gradle.kts      # Pre-configured Gradle build
├── settings.gradle.kts
├── gradle.properties     # Java 25 target
├── buildSrc/             # Custom Gradle plugin (optional)
├── hytale_server/
│   └── HytaleServer.jar
└── src/
    └── main/
        ├── kotlin/
        │   └── com/example/myplugin/
        └── resources/
            └── plugin.json
```

Key features:
- Shadow JAR packaging
- Automatic copy to plugins folder
- Kotlin 2.3.0 configured
- Java 25 compatibility
