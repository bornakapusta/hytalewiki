title: Common.ui Reference
description: Reference for shared UI variables, styles, and components in Common.ui
---

`Common.ui` provides the foundational variables, styles, and component templates used across all UI files.

## Importing

```ui
$C = "../Common.ui";
// or from deeper paths:
$C = "../../Common.ui";
```

## Panels & Containers

### @Panel
Basic panel with background.

```ui
@Panel = Group {
  Background: (TexturePath: "Common/ContainerFullPatch.png", Border: 20);
};
```

### @Container
Standard page container with title and content areas.

| Slot | Description |
|------|-------------|
| `#Title` | Header section |
| `#Content` | Main content area |
| `#CloseButton` | Optional close button |

```ui
$C.@Container {
  @CloseButton = true;  // Enable close button
  Anchor: (Width: 600, Height: 400);

  #Title {
    $C.@Title { @Text = "My Title"; }
  }

  #Content {
    LayoutMode: Top;
    // Content here
  }
}
```

### @DecoratedContainer
Container with decorative top/bottom ornaments.

```ui
$C.@DecoratedContainer {
  Anchor: (Width: 700, Height: 500);
  // Same slots as @Container
}
```

### @PageOverlay
Semi-transparent background overlay for modals.

```ui
$C.@PageOverlay {
  LayoutMode: Middle;
  // Container goes here
}
```

### Button Components

| Component | Description |
|-----------|-------------|
| `@TextButton` | Primary button with text |
| `@SecondaryTextButton` | Secondary styled button |
| `@TertiaryTextButton` | Tertiary styled button |
| `@CancelTextButton` | Red destructive button |
| `@SmallSecondaryTextButton` | Smaller secondary button |
| `@SmallTertiaryTextButton` | Smaller tertiary button |

```ui
$C.@TextButton {
  @Text = "Primary Action";
  @Anchor = (Width: 200);
}

$C.@SecondaryTextButton {
  @Text = "Secondary";
}

$C.@CancelTextButton {
  @Text = "Delete";
}
```

## Input Fields

### @TextField
Text input field.

```ui
$C.@TextField #NameInput {
  @Anchor = (Width: 300);
  PlaceholderText: "Enter text...";
}
```

### @NumberField
Numeric input with optional formatting.

```ui
$C.@NumberField #CountInput {
  @Anchor = (Width: 100);
  Format: (MaxDecimalPlaces: 0, Step: 1);
}
```

### @DropdownBox
Selection dropdown.

```ui
$C.@DropdownBox #TypeSelector {
  @Anchor = (Width: 200);
}
```

### @CheckBox
Boolean toggle.

```ui
$C.@CheckBox #EnableOption {
  Value: true;
}
```

### @CheckBoxWithLabel
Checkbox with text label.

```ui
$C.@CheckBoxWithLabel {
  @Text = "Enable feature";
  @Checked = false;
}
```

```ui
}
```

Right-aligned button container for footers.

```ui

  $C.@TextButton { @Text = "Save"; }
```
