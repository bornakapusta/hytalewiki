Reusable component templates used across multiple UI pages.

## Common/TextButton.ui

**Purpose**: Styled text button with selection states
**Location**: `Common/TextButton.ui`

| Element ID | Type | Description |
|------------|------|-------------|
| `#Button` | TextButton | Main button element |

```ui
@LabelStyle = (Hovered: (Background: #000000(0.2)));

@SelectedLabelStyle = (
  Default: (LabelStyle: (RenderBold: true)),
  Hovered: (LabelStyle: (RenderBold: true), Background: #000000(0.2))
);

Group {
  LayoutMode: Left;

  TextButton #Button {
    Padding: (Full: 6);
    Style: @LabelStyle;
    FlexWeight: 1;
  }
}
```

## Common/ActionButton.ui

**Purpose**: Action button with keyboard binding display
**Location**: `Common/ActionButton.ui`

| Element ID | Type | Description |
|------------|------|-------------|
| `#Container` | Group | Main layout container |
| `#NameRight` | Label | Action name (right-aligned) |
| `#NameLeft` | Label | Action name (left-aligned) |
| `#BindingModifier1Container` | Group | First modifier key |
| `#BindingModifier2Container` | Group | Second modifier key |
| `#BindingContainer` | Group | Main key binding |
| `#BindingLabel` | Label | Key binding text |
| `#BindingIcon` | Group | Mouse icon (for click bindings) |

```ui
Group #Container {
  LayoutMode: Left;
  Anchor: (Height: 24);

  Label #NameRight {
    Style: @LabelStyle;
    Anchor: (Right: 10);
    Visible: false;
  }

  Group #BindingContainer {
    Anchor: (Height: 24);
    Background: @BindingButtonBackground;
    Padding: (Horizontal: 6);

    Label #BindingLabel {
      Text: "???";
      Style: @BindingLabelStyle;
    }
  }

  Label #NameLeft {
    Style: @LabelStyle;
    Anchor: (Left: 10);
    Visible: false;
  }
}
```

## CommandListPage.ui

**Purpose**: Full command browser with details panel
**Location**: `Pages/CommandListPage.ui`

| Element ID | Type | Description |
|------------|------|-------------|
| `#CommandList` | Group | Scrollable command list (250px) |
| `#CommandInfo` | Group | Command details panel |
| `#CommandName` | Label | Command name |
| `#CommandUsageLabel` | Label | Usage syntax |
| `#CommandDescription` | Label | Command description |
| `#AliasesSection` | Group | Command aliases |
| `#AliasesList` | Label | Alias list |
| `#PermissionSection` | Group | Permission info |
| `#PermissionLabel` | Label | Required permission |
| `#BackButton` | TertiaryTextButton | Return to parent |
| `#SendToChatButton` | SecondaryTextButton | Copy to chat |
| `#VariantsSection` | Group | Command variants panel |
| `#VariantsList` | Group | Variant cards container |
| `#SubcommandSection` | Group | Subcommands section |
| `#SubcommandCards` | Group | Subcommand cards container |
| `#ParametersSection` | Group | Parameters panel |
| `#RequiredArgumentsList` | Group | Required args list |
| `#OptionalArgumentsList` | Group | Optional args list |
| `#DefaultArgumentsList` | Group | Default args list |
| `#FlagArgumentsList` | Group | Flag args list |
| `#ArgumentTypesSection` | Group | Argument types panel |
| `#ArgumentTypesList` | Group | Type items container |

```ui
$C.@Container {
  Anchor: (Width: 1050, Height: 700);

  #Content {
    LayoutMode: Left;

    Group #CommandList {
      Anchor: (Width: 250);
      LayoutMode: TopScrolling;
      ScrollbarStyle: $C.@DefaultScrollbarStyle;
    }

    Group #CommandInfo {
      FlexWeight: 1;
      LayoutMode: TopScrolling;
      Padding: (Left: 10);

      Label #CommandName {
        Style: (RenderUppercase: true, RenderBold: true, FontSize: 17);
      }

      Label #CommandUsageLabel {
        Style: (FontSize: 13, TextColor: #7a9cc6);
      }

      // ... additional sections
    }
  }
}
```
