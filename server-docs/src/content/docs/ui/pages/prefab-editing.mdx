Components for prefab editing, browsing, and configuration management.

## PrefabEditorSettings.ui

**Purpose**: Main prefab editor configuration with multiple sub-pages
**Location**: `Pages/PrefabEditorSettings.ui`

| Element ID | Type | Description |
|------------|------|-------------|
| `#SavedConfigs` | Group | Saved configuration dropdown |
| `#RootDir` | Group | Root directory selector |
| `#PrefabPaths` | Group | Prefab paths input with browse |
| `#BrowseButton` | SecondaryTextButton | Open file browser |
| `#Recursive` | Group | Recursive search toggle |
| `#Children` | Group | Load children toggle |
| `#Entities` | Group | Load entities toggle |
| `#EnableWorldTicking` | Group | World ticking toggle |
| `#DesiredYLevel` | NumberInput | Y-level setting |
| `#BlocksBetweenPrefabs` | NumberInput | Spacing setting |
| `#WorldGenType` | Group | World gen type dropdown |
| `#Environment` | Group | Environment dropdown |
| `#GrassTint` | Group | Grass color picker |
| `#PasteAxis` | Group | Paste axis dropdown |
| `#AlignmentMethod` | Group | Alignment dropdown |
| `#RowSplitMode` | Group | Row split dropdown |
| `#LoadButton` | TextButton | Load prefabs action |
| `#SavePropertiesButton` | SecondaryTextButton | Save config |
| `#CancelButton` | SecondaryTextButton | Cancel action |

```ui
Group #PrefabPaths {
  LayoutMode: Left;

  @InputLabel { Text: %server.customUI.prefabEditorSettings.prefabPaths; }

  $C.@TextField #Input {
    FlexWeight: 1;
    MaxLength: 4096;
  }

  $C.@SecondaryTextButton #BrowseButton {
    @Anchor = (Width: 140);
    Text: %server.customUI.prefabEditorSettings.browse;
  }
}
```

### Sub-pages

The file contains multiple overlay pages:

- `#MainPage` - Primary settings form
- `#SaveConfigPage` - Save configuration dialog
- `#LoadingPage` - Loading progress with status and progress bar
- `#BrowserPage` - File browser for prefab selection

## PrefabSavePage.ui

**Purpose**: Prefab save dialog with name and path
**Location**: `Pages/PrefabSavePage.ui`

| Element ID | Type | Description |
|------------|------|-------------|
| `#NameInput` | TextField | Prefab name |
| `#PathInput` | TextField | Save path |
| `#SaveButton` | TextButton | Save action |
| `#CancelButton` | SecondaryTextButton | Cancel |

```ui

    Group #PathSection {

    }

```
