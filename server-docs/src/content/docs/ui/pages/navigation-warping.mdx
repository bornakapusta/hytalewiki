Components for managing warp points, world instances, and player navigation.

## WarpListPage.ui

**Purpose**: Searchable list of available warp points
**Location**: `Pages/WarpListPage.ui`

| Element ID | Type | Description |
|------------|------|-------------|
| `#WarpList` | Group | Scrollable warp entries |
| `#SearchInput` | CompactTextField | Warp search (via HeaderSearch) |

```ui
$C.@Container {
  Anchor: (Width: 600, Height: 700);

  #Title {
    Group {
      $C.@Title { @Text = %server.customUI.warpListPage.warps; }
      $C.@HeaderSearch {}
    }
  }

  #Content {
    LayoutMode: Left;

    Group #WarpList {
      FlexWeight: 1;
      LayoutMode: TopScrolling;
      ScrollbarStyle: $C.@DefaultScrollbarStyle;
    }
  }
}
```

## WarpEntryButton.ui

**Purpose**: Individual warp point button in list
**Location**: `Pages/WarpEntryButton.ui`

| Element ID | Type | Description |
|------------|------|-------------|
| `#Button` | Button | Clickable warp entry |
| `#Name` | Label | Warp name (bold) |
| `#World` | Label | World name (italic, dimmed) |

```ui
Button #Button {
  Padding: (Full: 6);
  LayoutMode: Left;
  Style: (Hovered: (Background: #000000(0.2)));

  Label #Name {
    Style: (RenderBold: true);
    FlexWeight: 1;
  }

  Label #World {
    Style: (TextColor: #ffffff(0.2), RenderItalics: true);
  }
}
```
