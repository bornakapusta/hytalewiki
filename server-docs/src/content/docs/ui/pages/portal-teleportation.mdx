Components for portal interactions, teleportation devices, and player transportation.

## Portals.ui

**Purpose**: Main portal interface with artwork, objectives, and summon button
**Location**: `Pages/Portals/Portals.ui`

| Element ID | Type | Description |
|------------|------|-------------|
| `#LeftPane` | Group | Portal artwork display with runic border |
| `#RightPane` | Group | Info panel with objectives and tips |
| `#Title0` | Label | Portal name display |
| `#ExplorationTimeText` | Label | Time limit display |
| `#ObjectivesList` | Group | Dynamic objective list |
| `#TipsList` | Group | Dynamic tips list |
| `#Pills` | Group | Tag pills container |
| `#SummonButton` | TextButton | Portal activation button |

```ui
Group #LeftPane {
  Anchor: (Width: 431, Height: 716 + 30, Left: 15, Right: 15);

  Group #Artwork {
    Background: (TexturePath: "Portals/DefaultArtwork.png");
    Anchor: (Width: 345, Height: 629);
  }
}
```

## Teleporter.ui

**Purpose**: Teleporter configuration with coordinates and warp management
**Location**: `Pages/Teleporter.ui`

| Element ID | Type | Description |
|------------|------|-------------|
| `#ErrorScreen` | Group | Error display section |
| `#FullSettings` | Group | Main settings container |
| `#BlockRelative` | CheckBoxWithLabel | Block-relative toggle |
| `#X`, `#Y`, `#Z` | Group | Coordinate inputs |
| `#Yaw`, `#Pitch`, `#Roll` | Group | Rotation inputs |
| `#WorldDropdown` | DropdownBox | World selector |
| `#WarpDropdown` | DropdownBox | Warp point selector |
| `#NewWarp` | TextField | New warp name input |
| `#SaveButton` | TextButton | Save action |

```ui
Group #X {
  LayoutMode: Top;

  Label { Text: "X"; }

  $C.@NumberField #Input {
    @Anchor = (Width: 60);
    Format: (MaxDecimalPlaces: 2, Step: 0.5);
  }

  $C.@CheckBoxWithLabel {
    @Text = %server.customUI.teleporter.relative;
  }
}
```
